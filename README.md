# Project Description
This program is a sigle-player terminal-based number guessing game. The computer randomly
chooses a secret number, and the player tries to guess it based on feedback.  After each guess,
the program tells the player if the guess was too high, too low, or correct.  The program also
tracks attempts, saves previous guesses, includes difficulty levels, and lets 
the player replay.

# How to Run the Program
1. Make sure Python is installed on your computer.
2. Download this file or the repository.
3. Open the file in a terminal.
4. Run the program. (CMPSC_132_FP3.py)
